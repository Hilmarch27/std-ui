## Usage

```tsx
import { VerticalLineList } from "@/components/ui/vertical-list";
```

<br />
```tsx
<VerticalLineList
  items={[
    {
      text: "Step 1",
      subItems: ["Substep 1"],
      icon: <User size={12} />,
    },
    {
      text: "Step 2",
      subItems: ["Substep 1", "Substep 2"],
      icon: <CheckIcon size={12} />,
    },
  ]}
/>
```

## Examples

### Without Icon and Sub Items

You can adding icons by passing the `Icon JSX` prop. By default it is optional.
